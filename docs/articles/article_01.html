Chapter 4:Reversing an Array

Introduction to the Problem

In this problem you have to make a function called reverseArray which takes one argument array.

The array must be reversed like the examples below:

hello = ['0', 'l', 'l', 'e', 'H'];
reverseArray(hello);
lick = ['k', 'c', 'i', 'l'];
reverseArray(lick);
